# Tech Stack: Eidon 🌌

## 1. Fase Actual (Prototipado / Taller 4)
- **Estructura:** HTML5 Semántico.
- **Estilos:** CSS3 nativo (usando variables definidas en el Design System).
- **Control de Versiones:** Git con flujo de ramas `main`.
- **Despliegue:** GitHub Pages (Planeado para esta fase).

## 2. Fase de Desarrollo (Frontend)
- **Framework:** [React.js](https://react.js.org/) con Vite.
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/) (Utility-first framework).
- **Iconos:** [Lucide React](https://lucide.dev/) para coherencia visual.
- **Gestión de Estado:** React Context API o Zustand.

## 3. Fase de Persistencia (Backend & Datos)
- **Base de Datos:** [Firebase Firestore](https://firebase.google.com/) (NoSQL para escalabilidad rápida).
- **Autenticación:** Firebase Auth (Google & Email/Password).
- **Hosting:** Firebase Hosting o Vercel.

## 4. Convenciones de Código
- **Nomenclatura:** BEM para CSS nativo; camelCase para JavaScript.
- **Idioma:** Código y comentarios en inglés (estándar profesional), pero interfaz de usuario en español.
- **Commits:** Seguir el estándar de *Conventional Commits* (ej: `feat:`, `fix:`, `docs:`, `chore:`).
