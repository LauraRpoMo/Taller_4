# Eidon  
**Tu bitácora personal de consumo mediático.**

Eidon es una plataforma diseñada para centralizar y organizar todo el contenido multimedia que consumes. Desde películas y series hasta podcasts y artículos, Eidon te permite registrar, calificar y analizar tus hábitos de entretenimiento en un solo lugar.

---

## Características (Features)
- **Registro Multiformato:** Clasifica contenido como Video, Audio o Lectura.
- **Estado de Progreso:** Marca lo que estás "Viendo", "Pendiente" o "Completado".
- **Sistema de Valoración:** Califica tus experiencias y añade notas personales.
- **Categorización Inteligente:** Filtros por género, formato y fecha.
- **Dashboard Estadístico:** (Próximamente) Visualiza cuánto tiempo dedicas a cada tipo de media.

---

## Tecnologías
Este proyecto está construido con un enfoque profesional y escalable:
* **Frontend:** [React.js](https://reactjs.org/) (Planeado)
* **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
* **Estructura:** Arquitectura de carpetas profesional.
* **Control de Versiones:** Git & GitHub.
